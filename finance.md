# Finance

Moduły finansowe platformy Intum - fakturowanie i paragony fiskalne.

## Spis treści

1. [Faktury (Invoices)](#faktury-invoices)

---

## Faktury (Invoices)

Fakturowanie obsługiwane jest przez system **Fakturownia** - zintegrowany z platformą Intum moduł do wystawiania i zarządzania dokumentami sprzedażowymi.

Pełna dokumentacja API faktur: **[github.com/fakturownia/API](https://github.com/fakturownia/API)**

---

## Paragony (eParagony)

Paragony fiskalne obsługiwane są przez system **eParagony** - moduł do wystawiania i zarządzania paragonami elektronicznymi.

Pełna dokumentacja API paragonów: **[github.com/e-paragony/api](https://github.com/e-paragony/api)**
